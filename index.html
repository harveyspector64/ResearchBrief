A Director's Framework for AI Screen Tests

Thank you for this opportunity. This framework translates a director's vision into a structured methodology for stress-testing AI video models. My approach treats each model as a nascent "digital filmmaker," testing not just for technical execution but for its potential as a true creative partner. Below is the proposed methodology, designed to be rigorous, insightful, and actionable.

Conceptual Demo Reel

1. Methodology & Workflow

Philosophy

Treat each model as a "Director of Photography." We provide the same scene direction (the prompt) and evaluate how they "shoot" it, testing for both technical execution and creative interpretation.

Step-by-Step Process

  1. Establish 'Controlled Set': Use a curated set of base images (Midjourney v6) for Img-to-Video and a core prompt for Text-to-Video to ensure consistency.
  2. Execute 'Shot List': Systematically input test case prompts into each model, meticulously labeling and organizing all generated clips.
  3. Review 'Dailies': Evaluate each output against our rubric, combining quantitative scores with qualitative "Director's Notes."
  4. Synthesize for 'Final Cut': Compile findings into a single, polished report.

Tool Stack

Platforms: Model A/B/C, Midjourney. Organization: Google Sheets. Presentation: Adobe Premiere, Loom, a final report document.

`[expression subtly shifts from joy to sadness]`

3. Evaluation Rubric

Prompt Adherence

Did the model understand the cinematic term (pan, dolly)? Did it capture the mood?

Cinematic Quality

Does it look like a "shot" or just a moving picture? Is composition strong?

Technical Fidelity

How severe are warping or flickering artifacts? Is motion smooth?

Temporal Consistency

Does a character's face remain stable? Do background objects morph?

Each metric is scored 1-5, supplemented by qualitative "Director's Notes".

4. Final Deliverable

The Final Research Report

The findings will be consolidated into a single, concise report. This document will provide a clear and comprehensive overview of each model's capabilities, including:

  • An Executive Summary with at-a-glance findings and recommendations.
  • A 1-2 minute "Highlight Reel" (video) showcasing the most telling successes and failures.
  • Key Side-by-Side Comparisons embedded to illustrate critical differences in performance.
