Artist-Led Evaluation Framework for AI Video Models

A research methodology by Brian Hightman

Introduction

As an artist working daily with AI video models, my approach to research is both intuitive and methodical. This framework reflects how I probe the creative and technical capacities of multi-modal tools through cinematic logic, style stress-testing, and iterative prompt design.

Methodology

  1. Define test themes: camera logic, style, physics
  2. Design prompt variants across models
  3. Generate clips & group by test type
  4. Score and annotate: visual, motion, consistency
  5. Compile findings with cinematic context

Evaluation Rubric

  • Temporal Coherence: 4/5 (slight flicker in shadows)
  • Camera Logic: 5/5 (dolly motion is clean)
  • Style Adherence: 3/5 (inconsistent color palette)

Timeline

  • Day 3: Analysis + visual layout
